### Overview
This paper presents a deep learning-based approach for the detection of ocular diseases using retinal fundus images. Ocular disorders affect a large population worldwide, and early diagnosis remains challenging in regions with limited access to ophthalmic specialists. The proposed system utilizes MobileNetV2- based transfer learning to classify retinal images into four categories: Cataract, Diabetic Retinopathy, Glaucoma, and Normal. The dataset used in this study consists of 4,217 labelled fundus images obtained from a publicly available Kaggle source. The methodology involves data preprocessing, image augmentation, and model training using a convolutional neural network architecture. The model is trained over multiple epochs with early stopping to prevent overfitting. Performance is evaluated using metrics such as accuracy, precision, recall, and loss analysis. Experimental results show that the proposed model achieves a peak validation accuracy of 82.56%, demonstrating effective generalization across multiple disease classes. The findings highlight the potential of lightweight deep learning models for automated eye disease screening and their applicability in resource-constrained environments. This approach can assist in early detection and support clinical decision-making processes.
